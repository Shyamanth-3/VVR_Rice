## 📁 Repository Structure

This repository contains the source code and assets for the **VVR Rice (India)** digital presence and operations.

```
VVR_Rice/
├── src/                  # React frontend source code
│   ├── components/       # Reusable React components
│   ├── pages/            # Route-based pages
│   ├── assets/           # Images and media files
│   ├── hooks/            # Custom React hooks
│   ├── utils/            # Utility functions
│   ├── App.jsx
│   ├── main.jsx
│   └── index.css
├── server/               # Express.js backend API
│   ├── server.js
│   └── package.json
├── ETC/                  # Utilities and tools
│   └── convert.py
├── package.json          # Frontend dependencies
├── vite.config.js        # Vite configuration
├── tailwind.config.js    # Tailwind CSS configuration
└── Assets/               # Company branding & media (excluded from git)
```

## 🌐 Website Project

The main project is the **VVR Rice Official Website**, a modern, responsive platform showcasing:

- **Company Heritage**: 60+ years of rice milling excellence
- **Product Portfolio**: Premium rice varieties (Sona Masoori, FRK, MTU 1061)
- **Business Solutions**: B2B partnerships and bulk orders
- **Customer Engagement**: Contact form and inquiry management
